# SpaceX-Falcon-Heavy

SpaceX Falcon Heavy interactive page.

The Desktop-first approach was chosen.

Technologies:
* HTML5 & CSS3, SASS(SCSS), JavaScript(ES6). 

Libraries:
* Vidbg
* Rellax
* AOS

[Demo](https://nazar197.github.io/SpaceX-Falcon-Heavy/)
